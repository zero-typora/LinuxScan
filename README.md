# LinuxScan

linux 自动化应急信息收集工具

支持联动微步分析外联IP是否为恶意IP，自动话webshell扫描，自动化分析恶意进程，文件上传修改，自动化日志分析

![image-20240320171749589](image-20240320171749589.png)

## 使用方法

```shell
./linuxScan
```
